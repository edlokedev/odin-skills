# project-audit

Purpose: audit a project.

## Flow

1. Read `project.yaml`.
2. Read `status.md`.
3. Read `decisions.md`.
4. Read Graphify report if available.
5. Inspect source files.
6. Produce audit with risks, gaps, and next actions.
