# Attribution

This repository is a curated bundle of agent skills assembled for use across multiple
coding-agent platforms. Many skills originate from third parties and remain the property
of their respective authors under their original licenses.

Where a skill ships its own license, that file is preserved inside the skill's folder
(e.g. `skills/code-review-and-quality/awesome-skills/LICENSE`). Those licenses govern
the corresponding skill regardless of this repository's top-level `LICENSE`.

Known upstreams include, among others:

- **Matt Pocock** — `grill-me`, `grill-with-docs`, `tdd`, and related plan-hardening
  skills (MIT).
- **Anthropic** — design/`impeccable` suite, document skills, and other first-party
  Claude skills.
- **Stitch** — `stitch-design`, `stitch-loop`, `react-components`, `remotion`,
  `design-md`, and related design-tooling skills.
- **shadcn/ui** and other component-library projects — `shadcn-ui`.

If you are an author and want attribution corrected or a skill removed, open an issue.

The top-level `LICENSE` covers only the original contributions in this repository
(packaging, manifests, and Odin-authored skills). It does not relicense third-party
material.
